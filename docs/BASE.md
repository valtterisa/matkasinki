# Vacation Planner — Feature Base

- **Onboarding:** a detailed multiple choice questionnaire.
- **Trip Planner:** dates, budget, vibe → detailed and full itinerary.
- **Out-of-Office:** auto-reply, triage, and summarise while the user is away (demo with Resend).
- **Activity Finder:** weather + location + mood → plans for the day.
- **Packing List:** destination + weather + trip type → what to bring; also updates daily.
- **Reading Curator:** suggest a reading list from personality-test results.
- **Home Sitter:** run automation (demo with an MCP server).
- **Budget Tracker:** log expenses, easily convert currencies, track the trip.
- **Local Guide (IMPORTANT):** find restaurants and hidden gems within budget, and come up with challenges.
- **Team Building:** challenges done and places visited add to resources that build the user's own football team.
